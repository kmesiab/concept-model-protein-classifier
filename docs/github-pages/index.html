Quick Start

Get started with a simple API call:

  -H "X-API-Key: your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "sequences": [
      {"id": "protein1", "sequence": "MKVLWAASLLLLASAARA"}
    ]
  }'
